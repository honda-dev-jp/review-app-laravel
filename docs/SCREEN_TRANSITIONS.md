# 画面遷移

## このドキュメントの目的

このドキュメントでは、映画レビューアプリ Laravel移植版の画面遷移を整理する。

初期移植フェーズでは、既存スクラッチ版の最低限機能をLaravelへ移植するため、ゲスト・会員・共通画面を中心に整理する。

管理者画面は後続フェーズで検討する。

## 画面区分

| 区分 | 概要 |
|---|---|
| 共通画面 | ゲスト・会員・管理者が閲覧できる公開画面 |
| ゲスト画面 | ログインしていないユーザーが利用する画面 |
| 会員画面 | ログイン済みユーザーが利用する画面 |
| 管理者画面 | 作品情報を管理する画面。初期移植フェーズでは対象外 |

## 初期移植フェーズの画面一覧

### 共通画面

| 画面 | 概要 | 認証 |
|---|---|---|
| 作品一覧画面 | 登録済み作品を一覧表示する | 不要 |
| 作品詳細画面 | 作品詳細、レビュー、レビュー返信、星評価を表示する | 不要 |

### ゲスト画面

| 画面 | 概要 | 認証 |
|---|---|---|
| 会員登録画面 | 新規会員登録を行う | 不要 |
| ログイン画面 | 既存会員がログインする | 不要 |

### 会員画面

| 画面 | 概要 | 認証 |
|---|---|---|
| プロフィール編集画面 | ユーザー名やプロフィール情報を編集する | 必要 |
| 本人のレビュー一覧画面 | 自分が投稿したレビューを一覧表示する | 必要 |
| 会員退会確認画面 | アカウント退会前の確認を行う | 必要 |

## 共通ナビゲーション

共通ヘッダーには、ログイン状態に応じたナビゲーションを表示する。

### ゲスト表示

- 作品一覧
- ログイン
- 会員登録

### 会員表示

- 作品一覧
- プロフィール編集
- 本人のレビュー一覧
- ログアウト

退会は危険操作のため、共通ナビゲーションには直接表示しない。

退会導線はプロフィール編集画面配下に配置する。

### レスポンシブ対応

PC表示ではヘッダーにナビゲーションを表示する。

スマホ表示ではハンバーガーメニュー内にナビゲーションを表示する。

ナビゲーションの表示切り替えは Blade の `@auth` / `@guest` を使用する。

ログイン必須ページへのアクセス制御は `auth` ミドルウェアで行う。

## 共通画面の遷移

```text
作品一覧画面
  └─ 作品リンククリック
      ↓
      作品詳細画面
        ├─ レビュー表示
        ├─ レビュー返信表示
        └─ 星評価表示
```

## ゲストの画面遷移

```text
共通ナビゲーション
  ├─ 作品一覧画面
  ├─ ログイン画面
  └─ 会員登録画面

作品一覧画面
  └─ 作品リンククリック
      ↓
      作品詳細画面
        ├─ レビュー表示
        ├─ レビュー返信表示
        └─ 星評価表示

会員登録画面
  ↓
登録完了後
  ↓
作品一覧画面
```

レビュー・評価投稿やレビュー返信投稿などの会員操作は、会員の画面遷移に整理する。

## 会員の画面遷移

```text
ログイン画面
  ↓
ログイン成功
  ↓
作品一覧画面

作品一覧画面
  └─ 作品リンククリック
      ↓
      作品詳細画面
        ├─ レビュー・評価投稿
        └─ レビュー返信投稿

共通ナビゲーション
  ├─ プロフィール編集画面
  │   └─ 会員退会確認画面
  ├─ 本人のレビュー一覧画面
  │   └─ レビュー削除
  └─ ログアウト
```

ログイン成功後は、ダッシュボード画面ではなく作品一覧画面へ遷移する。

レビュー削除は、本人のレビュー一覧画面からのみ実行できるものとする。

作品詳細画面ではレビュー削除導線を表示しない。

## 認証が必要な画面・操作

| 画面・操作       | 認証 | 認可          |
| ----------- | -- | ----------- |
| レビュー・評価投稿 | 必要 | 会員のみ        |
| レビュー削除      | 必要 | 自分のレビューのみ   |
| レビュー返信投稿    | 必要 | 会員のみ        |
| プロフィール編集    | 必要 | 自分のプロフィールのみ |
| 本人のレビュー一覧表示 | 必要 | 自分のレビューのみ   |
| 会員退会        | 必要 | 自分のアカウントのみ  |

## 後続フェーズで検討する画面

### お問い合わせ画面

| 画面 | 概要 |
|---|---|
| お問い合わせフォーム画面 | ゲスト・会員が問い合わせを送信する |
| お問い合わせ送信完了画面 | 問い合わせ送信後の完了メッセージを表示する |

### 管理者画面

| 画面 | 概要 |
|---|---|
| 管理者作品一覧画面 | 管理者が作品一覧を確認する |
| 管理者作品詳細画面 | 管理者が作品詳細を確認する |
| 作品登録画面 | 管理者が作品を登録する |
| 作品編集画面 | 管理者が作品を編集する |
| 作品削除確認画面 | 管理者が作品削除を確認する |
| お問い合わせ管理画面 | 管理者が問い合わせ内容を確認・対応する |
| お問い合わせ詳細画面 | 管理者が問い合わせ内容の詳細を確認する |

### 外部API連携画面

| 画面           | 概要                  |
| ------------ | ------------------- |
| TMDB検索画面     | TMDB APIから映画情報を検索する |
| TMDB作品取り込み画面 | APIから取得した映画情報を登録する  |

## 補足

このドキュメントでは、画面間の流れを整理する。

具体的なURLやHTTPメソッドは `docs/ROUTES.md` に整理する。

DB設計は `docs/DATABASE.md` に整理する。

認証・認可の詳細は `docs/SECURITY.md` に整理する。
