# 機能一覧

## このドキュメントの目的

このドキュメントでは、映画レビューアプリ Laravel移植版で実装する機能を一覧化する。

初期移植フェーズでは、既存スクラッチ版の最低限機能をLaravelへ移植することを優先する。

管理者機能、外部API連携、レビュー編集などは後続フェーズで検討する。

## 機能区分

| 区分 | 概要 |
|---|---|
| 共通機能 | ゲスト・会員・管理者が利用できる公開画面系の機能 |
| ゲスト機能 | ログインしていないユーザーが利用できる機能 |
| 会員機能 | ログイン済みユーザーが利用できる機能 |
| 管理者機能 | 作品情報を管理する機能 |
| 外部API連携 | TMDB APIなど外部サービスと連携する機能 |

## 初期移植フェーズ

### 共通機能

| 機能 | 概要 | 優先度 |
|---|---|---|
| 作品一覧表示 | 登録済み作品を一覧表示する | 高 |
| 作品一覧ページネーション | 作品一覧をページ単位で表示する | 高 |
| 作品詳細表示 | 作品の詳細情報を表示する | 高 |
| レビュー表示 | 作品に投稿されたレビューを表示する | 高 |
| レビュー返信表示 | レビューに紐づく返信を表示する | 中 |
| 星評価表示 | 作品の平均評価や評価件数を表示する | 高 |

### ゲスト機能

| 機能 | 概要 | 優先度 |
|---|---|---|
| 会員登録 | Laravel Breezeをベースに会員登録できるようにする | 高 |
| ログイン | Laravel Breezeをベースにログインできるようにする | 高 |

### 会員機能

| 機能 | 概要 | 優先度 |
|---|---|---|
| ログアウト | Laravel Breezeをベースにログアウトできるようにする | 高 |
| レビュー・評価投稿 | 作品にレビュー本文と評価を投稿できるようにする | 高 |
| レビュー削除 | 自分が投稿したレビューを削除できるようにする | 高 |
| レビュー返信投稿 | レビューに対して返信を投稿できるようにする | 中 |
| プロフィール編集 | ユーザー名やプロフィール情報を編集できるようにする | 中 |
| 本人のレビュー一覧表示 | 自分が投稿したレビューを一覧表示する | 中 |
| レビュー履歴ページネーション | 自分のレビュー一覧をページ単位で表示する | 中 |
| 会員退会 | ユーザーが自分のアカウントを退会できるようにする | 中 |

## 後続フェーズで対応を検討する範囲

### 共通機能

| 機能 | 概要 | 優先度 |
|---|---|---|
| お問い合わせフォーム | ゲスト・会員が問い合わせを送信できるようにする | 低 |

### 会員機能

| 機能 | 概要 | 優先度 |
|---|---|---|
| レビュー編集 | 自分が投稿したレビュー本文と評価を編集できるようにする | 低 |
| レビュー返信編集 | 自分が投稿したレビュー返信を編集できるようにする | 低 |
| レビュー返信削除 | 自分が投稿したレビュー返信を削除できるようにする | 低 |
| お気に入り機能 | ログイン済みユーザーが作品をお気に入り登録できるようにする | 低 |

お気に入り機能を追加した場合、退会ユーザーのお気に入りデータは削除する方針とする。

### 管理者機能

| 機能 | 概要 | 優先度 |
|---|---|---|
| 作品一覧表示 | 管理画面で作品一覧を表示する | 低 |
| 作品詳細表示 | 管理画面で作品詳細を表示する | 低 |
| 作品登録 | 管理者が作品情報を登録できるようにする | 低 |
| 作品編集 | 管理者が作品情報を編集できるようにする | 低 |
| 作品削除 | 管理者が作品情報を削除できるようにする | 低 |
| 画像アップロード | 作品画像をアップロードできるようにする | 低 |
| お問い合わせ管理 | 送信された問い合わせ内容を管理画面で確認・対応できるようにする | 低 |

### 外部API連携

| 機能 | 概要 | 優先度 |
|---|---|---|
| TMDB API連携 | TMDB APIから映画情報を取得する | 低 |
| 作品情報の自動取得 | 映画タイトル、概要、公開日などを自動取得する | 低 |
| ポスター画像取得 | TMDB APIからポスター画像を取得する | 低 |

### テスト・運用

| 機能 | 概要 | 優先度 |
|---|---|---|
| Featureテストの追加 | 主要画面や投稿処理のFeatureテストを追加する | 低 |
| 主要機能の自動テスト整備 | レビュー投稿、削除、認可などの自動テストを整備する | 低 |
| トラブルシューティング整理 | 開発中に発生した問題と対応方法を整理する | 低 |
| デプロイ手順整理 | 本番反映に向けた手順を整理する | 低 |

## 認証・認可が必要な機能

| 機能 | 認証 | 認可 |
|---|---|---|
| レビュー・評価投稿 | 必要 | 会員のみ |
| レビュー削除 | 必要 | 自分のレビューのみ |
| レビュー返信投稿 | 必要 | 会員のみ |
| プロフィール編集 | 必要 | 自分のプロフィールのみ |
| 本人のレビュー一覧表示 | 必要 | 自分のレビューのみ |
| 会員退会 | 必要 | 自分のアカウントのみ |

4. レビュー表示
5. 星評価表示
6. Breeze認証の確認
7. レビュー・評価投稿
8. レビュー削除
9. プロフィール編集
10. 本人のレビュー一覧表示
11. レビュー履歴ページネーション
12. レビュー返信表示
13. レビュー返信投稿
14. 会員退会

## 補足

このドキュメントでは、機能の一覧と優先度を整理する。

画面ごとの遷移は `docs/SCREEN_TRANSITIONS.md` に整理する。

DB設計は `docs/DATABASE.md` に整理する。

ルーティング設計は `docs/ROUTES.md` に整理する。

認証・認可の詳細は `docs/SECURITY.md` に整理する。
